# Auction Bot [Discord.js]
 บอทประมูลสินค้า ทำโดย discord.js
 
## Installation
install
 ```shell
 npm install discord.js@12
 npm install ms
 npm install fs
 npm install quick.db
 ```
Put Your Bot Token in config.json

## Usage
 ```
 +start <name> <startprice> <minimumbid> <autowin> <time>
 +bid <price>
 +dlct <categoryid>
 ```
